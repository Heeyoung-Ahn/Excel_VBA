Log In UserForm

1. 프로그램의 내용
	- 엑셀 유저폼을 이용하여 DB  로그인
	- 리본메뉴에 추가기능으로 메뉴 부착
	- 로그인 시 체크항목
		1) 사용자 이름: Excel Application User Name으로 체크
		2) 비밀번호: 최초 접속 시 비밀번호 설정, 이후 접속 시 비밀번호 체크
		3) 프로그램 버전 체크
		4) 사용자 IP 체크
2. Create DB Table
	- common.users
	- common.logs
	# code: common.sql
3. DB사용자
	- common
	- 권한: logs - insert / users - select, update
	# code: grant_privileges_to_common_user.sql
3. 사용하는 Sub, Function code
	- a.Common.bas: Public Constant, Public Variable, DB연결, 기타 공통 코드
	- a.ErrHandler.bas: 에러 발생 시 보고(MsgBox)
	- a.WrigeLog.bas: 로그 기록
	- a.Ribbon.bas: 추가기능 리본메뉴
	- s_setGlobalVariant.bas: 로그인 시 전역변수 설정
	- fn_GetLocalIPaddress.bas: 사용자 PC IP주소 조회
	- fn_InputboxPW.bas: InputBox 입력 시 보안처리
	- fn_sha512.bas: DB에서 SHA-512방식으로 단방향 암호화 된 내용과 VBA에서 입력된 내용을 비교하기 위한 VBA 코드
4. Form code
	- f_login.frm: UserForm 파일
	- f_login.frx: UserForm의 디자인 파일
5. 완성본(명령단추 추가기능 형태)
	- login.xlam
